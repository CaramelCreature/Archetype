The Science Stuff

How we turn your chaotic chats into rigorous data.

Methodology

This project leverages NLP techniques to solve a novel, campus-specific problem: understanding IITD student archetypes from natural language. By combining classification (archetype detection) with regression (statblock generation), we create a rich, personalized profile beyond simple labels.

1

Data Ingestion

We parse standard WhatsApp export files (`_chat.txt`), handling both iOS and Android date formats. Regex patterns extract timestamps, senders, and message content while discarding media notifications.

2

Feature Engineering

We analyze the text for specific linguistic markers:

  • Keyword Density: Frequency of domain-specific terms (e.g., "DepC", "sutta", "deadline").
  • Stylistic Features: Emoji usage, message length, and capitalization patterns.
  • Interaction Metrics: Reply latency and activity distribution (e.g., late-night owls vs. morning birds).
3

Classification Model

Each user is scored against 24 distinct archetype profiles. We use a weighted heuristic model that calculates cosine similarity between the user's feature vector and the archetype's centroid.

4

Statblock Regression

Beyond the label, we generate a 5-point statblock (Focus, Social, Sleep, Ambition, Flexibility) based on the primary archetype's base stats + individual user deviations.

Privacy Note

All processing happens client-side in your browser. Your chat data never leaves your device.
